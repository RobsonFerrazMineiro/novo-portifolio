Jogo do Número Secreto

O Jogo do Número Secreto é um jogo interativo onde o jogador deve adivinhar um número aleatório dentro de um intervalo pré-definido. A cada tentativa, o jogo informa se o número digitado é maior ou menor que o número secreto, ajudando o jogador a se aproximar da resposta correta.

Principais Características do Jogo:
  • O jogo gera um número aleatório no início da partida.
  • O jogador insere um palpite por meio de comandos de voz ou digitando.
  • O jogo responde com dicas como "o número é maior" ou "o número é menor".
  • O objetivo é adivinhar o número correto no menor número de tentativas possível.
    • HTML
    • CSS
    • JavaScript
    • Web Speech API

    Alurabook

    O Alurabook é um projeto de e-commerce focado na venda de livros, inspirado na plataforma da Alura para aprendizado de tecnologia e desenvolvimento.

    Principais Características do Alurabook:
  • Catálogo de Livros – Apresenta uma lista de livros disponíveis para compra, organizados de forma acessível e intuitiva.
  • Carrinho de Compras – Possível funcionalidade para adicionar e visualizar os itens antes da compra.
  • Busca e Filtros – Permite encontrar livros por categorias ou palavras-chave.
  • Responsividade – Interface adaptada para diferentes dispositivos, tornando a experiência de navegação agradável.
    • HTML
    • CSS
    • Swiper API

    Vidflow

    O VidFlow é uma plataforma de streaming de vídeos que permite aos usuários explorar, assistir e enviar vídeos. O projeto inspirado em plataformas como YouTube e Vimeo, oferecendo uma experiência moderna e interativa para consumo de conteúdo audiovisual.

    Principais Características do Vidflow:
  • Reprodução de Vídeos – Permite assistir a vídeos diretamente no navegador, com controles de player intuitivos.
  • Sistema de Busca e Descoberta – Possível funcionalidade para encontrar vídeos por categorias, palavras-chave ou recomendações.
  • Perfis de Usuários – Pode incluir login, cadastro e personalização da experiência do usuário.
  • Design Responsivo – Interface adaptada para desktop e dispositivos móveis.
    • HTML
    • CSS
    • JavaScript
    • Node.js
    • Vite
    • Consumo de API
